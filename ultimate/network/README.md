# Unix Network

## Network Interface

--------------------------------------------------

### Commands

1. **ip**
    - ip addr
    - ip link

1. **nc**
    - nc -l `8080`
    - nc -l `127.0.0.1` `8080`
1. **netstat**
    - netstat -tulnp
    - netstat -rn #routing table
    - netstat -i `eth0`

## Routing Table

--------------------------------------------------

### Commands

1. **ip**
    - ip route get
    - ip route list table
